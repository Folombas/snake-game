# Snake Game

Классическая игра "Змейка" реализованная на Lua с использованием фреймворка Love2D.

## Управление

- Стрелки: Движение змейки
- Enter: Старт/рестарт игры
- P: Пауза
- Escape: Выход

## Особенности

- Система рекордов
- Достижения
- Минималистичный дизайн
- Звуковые эффекты

## Зависимости

- Love2D 11.3+
